# Pig-Game.github.io
A game between 2 Players  played using rolling Dice 
# CONDITIONS : 
# 1) If player enounters '1' on Dice ,he will lose the temporary score he scored in the current round 
# 2) Player can 'HOLD' his temporary score when he wishes he got good enough in each round instead of being loosing that when he encounters '1'
# 3) The Player who reaches total target score of '>=100' first will be the winner of this game
# Thia game totally depends on Luck factor (:
